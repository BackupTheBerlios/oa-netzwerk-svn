Ändern von Regelsätzen

Auf dieser Seite können Regelsätze geändert werden. Dabei kann, wenn gewünscht, der Name per Eingabe geändert und mittels Klick auf die zugehörigen Regeln gewählt werden, welche Regeln zum Regelsatz gehören sollen.





Name des Regelsatzes (Help)
Bitte hier den Namen eingeben, den der neu erstellte Regelsatz erhalten soll.


Zugehörige Regeln (Help)
Bitte die gewünschten Regeln durch Klick auf CheckBox an- oder abwählen.
